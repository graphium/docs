## Overview

The AI Assistant chat interface lets you type natural language questions about your vault data and receive richly formatted responses in real time. Responses can include plain text, bullet points, bold text, tables, charts, and structured insights — making complex data easy to read at a glance.

## When to Use This

Use this whenever you want to ask questions about your data, request analysis, or get summaries without navigating the data grid manually.

## Prerequisites

- You must have access to the vault
- At least one AI Assistant must be configured in the vault

## Step-by-Step Instructions

### Step 1: Open the Chat Input

Open a vault and the **Home** page loads automatically. The input bar at the top of the page shows the placeholder text *"Ask a question about your data..."*

### Step 2: Type Your Message

Click the input bar and type your question or request. For example:

- *"Show me today's case volume by facility"*
- *"Which providers had the most cases this week?"*
- *"Give me a summary of turnover times by OR room"*

### Step 3: Send the Message

Click the **send** button (arrow icon) on the right side of the input bar, or press **Enter**. The message is submitted and the AI begins generating a response.

### Step 4: Read the Response

The AI response appears in the chat area, labeled **AI Assistant**, and can include several content types:

- **Headers** — for organized multi-section responses (e.g., *Cases Overview*, *Key Insights*)
- **Bold text** — for emphasis and key data points
- **Bullet points** — for lists and summaries
- **Tables** — for structured data comparisons (e.g., a ranked list of slowest turnover times)
- **Charts** — bar charts, comparisons, and other visualizations rendered inline in the response
- **Key Insights callouts** — a structured summary of observations and recommendations at the end of a response

Each message displays a timestamp showing when it was sent or received.

### Step 5: Continue the Conversation

Ask follow-up questions in the same conversation — the AI maintains context from previous messages in the thread.

## What to Expect

The AI responds based on the data it has been given access to via the Chat Context configuration. Use the **⚙** button in the input bar to adjust which tables, tools, and connections the AI can access. The **Choose assistant** icon (leftmost in the input bar) lets you switch to a different assistant mid-session.

## Common Issues & Troubleshooting

| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| AI response does not include expected data | The relevant table may not be enabled in Chat Context | Click the ⚙ button and enable the table under the **Tables** tab |
| AI response does not include charts | **Chart Generation** is not enabled in Chat Context | Open Chat Context, click the **Tools** tab, and toggle **Chart Generation** on |
| Send button is greyed out | The input field is empty | Type a message before clicking Send |
