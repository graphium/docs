## Overview

Each table in Graphium AI has a set of configurable settings accessible from the **Settings** tab in the table view. From here you can rename the table, update its display names, assign an icon, and more.

> **Note:** The Settings tab is only visible to **ADMIN** and **BUILDER** users. CONTRIBUTOR and VIEWER users cannot access these settings.

---

## Renaming a Table

The table name can be updated directly from the table header without leaving the page.

1. Open the table you want to rename.
2. Click the **pencil icon (✏️)** to the right of the table name in the header.
3. The name becomes an editable text field. Type the new name.
4. Press **Enter** to save, or **Escape** to cancel.

The name updates instantly in the header and sidebar. Clicking outside the field also saves the name.

---

## Editing Table Display Names

Each table has a **Singular Name** and **Plural Name** that appear throughout the interface — for example, in the "Add [Name]" button and table headers.

1. Open the table and click the **Settings** tab.
2. In the **Table Names** section, update the **Singular Name** and **Plural Name** fields.
3. Click or tab out of the field to save automatically.

New tables default to **Item** (singular) and **Items** (plural). Both fields are required.

---

## Selecting a Table Icon

Each table can be assigned an icon that appears next to its name in the sidebar.

1. Open the table and click the **Settings** tab.
2. In the **Table Icon** section, click any of the 7 available icons to select it.

The selected icon is highlighted with a teal border and updates in the sidebar immediately. Click **Default** to remove a custom icon.

## Troubleshooting

| Issue | Likely Cause | What to Do |
|-------|-------------|------------|
| Pencil icon not visible next to table name | You have a CONTRIBUTOR or VIEWER role | Contact your vault Admin to request a higher role |
| Settings tab is not visible | You have a CONTRIBUTOR or VIEWER role | Contact your vault Admin — only ADMIN and BUILDER users can access table settings |
| Name reverts after saving | Network or save error | Check your connection and try again |
| Display name not updating in the interface | Page may need refreshing | Refresh the page — the change is saved and will display correctly |
