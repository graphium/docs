## Overview

Each table in a vault can be independently enabled or disabled for Contributor Mode. When enabled, the table becomes visible to users in Contributor Mode. Tables that are disabled are hidden from that view entirely.

This setting is controlled by the **Enable for Contributor Mode** checkbox in the **Settings** tab of the table view. Changes take effect immediately.

> **Note:** The Settings tab is only visible to **ADMIN** and **BUILDER** users. CONTRIBUTOR and VIEWER users cannot see or change this setting.

---

## Builder Mode vs Contributor Mode

Graphium AI has two sidebar modes, toggled using the **Builder / Contributor** switch at the bottom of the vault sidebar.

**Builder Mode** is the default for ADMIN and BUILDER users. It shows:
- All tables listed in the sidebar regardless of their Contributor Mode setting
- Full access to Data, Schema, and Settings tabs on every table
- The ability to add, rename, and configure tables

**Contributor Mode** is a simplified view designed for users who only need to view and edit data without modifying the vault structure. In Contributor Mode:
- The TABLES section in the sidebar is hidden unless a table has been explicitly enabled for Contributor Mode
- Only tables with **Enable for Contributor Mode** checked appear in the sidebar
- The Schema and Settings tabs are not accessible

ADMIN and BUILDER users can switch between modes at any time using the toggle — this is useful for previewing exactly what Contributor-role users see.

---

## How to Enable a Table for Contributor Mode

1. Open the table you want to make available in Contributor Mode.
2. Click the **Settings** tab in the top-right area of the table view.
3. Check the **Enable for Contributor Mode** checkbox.

The change saves immediately. The table will now appear in the sidebar when any user switches to Contributor Mode.

---

## How to Disable a Table for Contributor Mode

1. Open the table and click the **Settings** tab.
2. Uncheck the **Enable for Contributor Mode** checkbox.

The table is removed from the Contributor Mode sidebar view immediately. ADMIN and BUILDER users can still access the table normally in Builder Mode.

## Troubleshooting

| Issue | Likely Cause | What to Do |
|-------|-------------|------------|
| Settings tab is not visible | Your vault role is CONTRIBUTOR or VIEWER | Contact your vault Admin — only ADMIN and BUILDER users can access table settings |
| Table does not appear in Contributor Mode after enabling | Sidebar may not have refreshed | Wait a moment and refresh the page if it persists |
| Table still appears in Contributor Mode after disabling | Sidebar may not have refreshed | Wait a moment and refresh the page if it persists |
