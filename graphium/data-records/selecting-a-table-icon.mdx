## Overview

Each table in Graphium AI can be assigned an icon that appears next to its name in the sidebar navigation. Selecting an icon makes it easy to visually distinguish tables at a glance — especially useful in vaults with many tables. The icon is configured from the **Settings** tab of the table and updates in the sidebar immediately when selected.

---

## Opening the Table Icon Selector

1. Open a vault and select the table you want to configure from the left sidebar.
2. Click the **Settings** tab in the top-right corner of the table view.
3. Scroll to the **Table Icon** section, below the Enable for Contributor Mode checkbox.

> **Note:** The Settings tab is only visible to users with the **Admin** or **Builder** role. Contributors and Viewers do not have access to table settings.

---

## Selecting an Icon

The Table Icon section displays a row of seven options with the helper text *"Select an icon to represent this table in the navigation bar."*

Click any option to select it. The selected option is highlighted with a **teal rounded border**. The table's sidebar entry updates immediately to display the chosen icon.

### Available Options

| Option | Description |
|--------|-------------|
| Default | Removes any custom icon and reverts the table to its default appearance |
| Document | A page or file — suitable for general record tables |
| Clock | A clock face — suitable for time-based or scheduled data |
| Shield Check | A shield with a checkmark — suitable for compliance or verification tables |
| Database | A cylinder stack — suitable for data-heavy or reference tables |
| Bolt | A lightning bolt — suitable for activity or automation-related tables |
| People | A group of figures — suitable for person or contact tables |

Only one option can be active at a time. Clicking a new option immediately replaces the previous selection.

---

## How the Icon Appears

After selecting an icon, it appears in two places:

- **Left sidebar** — next to the table name in the TABLES list, visible from any view within the vault.
- **Table header** — the icon is reflected in the table's visual identity across the application.

To remove a custom icon and revert to the default appearance, click the **Default** option. If no custom icon has been set, the table displays a default appearance in the sidebar automatically.

---

## Permissions

| Role | Can Select Table Icon |
|------|-----------------------|
| Admin | ✅ |
| Builder | ✅ |
| Contributor | ❌ |
| Viewer | ❌ |

The Settings tab — including the Table Icon selector — is not visible to Contributor or Viewer roles. Externally managed tables (those displaying a "Managed by" badge) also have Settings disabled and cannot have their icon changed from within Graphium AI.
