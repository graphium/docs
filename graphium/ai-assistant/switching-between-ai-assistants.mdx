## Overview

Each vault can have multiple AI Assistants configured — each with a different focus, data access, and set of capabilities. You can switch between assistants at any time using the assistant selector in the chat input bar.

## When to Use This

Use this when you want to ask a different type of question that is better suited to a specialized assistant — for example, switching from a general assistant to a Data Analyst for in-depth reporting, or to an Operations Helper for workflow questions.

## Prerequisites

- The vault must have more than one AI Assistant configured
- You must have access to the AI Assistant section of the vault

## Step-by-Step Instructions

### Step 1: Open the Assistant Selector

In the chat input bar at the bottom of the screen, click the **chat bubble icon** (leftmost icon). A popup appears showing the available assistants for this vault.

### Step 2: Review the Available Assistants

The **Assistant** panel lists all configured assistants, for example:

| Assistant | Focus |
|-----------|-------|
| **General** | Broad questions about your vault data |
| **Data Analyst** | In-depth data analysis, reporting, and metrics |
| **Operations Helper** | Operational workflows and process questions |
| **Compliance Monitor** | Compliance checks and regulatory monitoring |

The currently active assistant is shown with a checkmark (✓). A **Manage** link at the top of the panel opens assistant configuration settings.

### Step 3: Select an Assistant

Click the assistant you want to use. The panel closes and the input bar updates to reflect the selected assistant (e.g., *"Chatting with Data Analyst"*).

### Step 4: Send Your Message

Type your question in the input bar and send it. The selected assistant will respond based on its configured capabilities and data access.

## What to Expect

Each assistant may have different access to tables, tools, and connections. Switching assistants mid-conversation starts using the new assistant for all subsequent messages. Previous messages in the conversation remain visible for context.

## Common Issues & Troubleshooting

| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| Only one assistant appears in the selector | Only one assistant is configured for this vault | Contact your vault Admin to configure additional assistants |
| The assistant I need is not in the list | It may not be configured for this vault | Contact your vault Admin to add the assistant |
