## Overview

The vault sidebar includes a **Builder / Contributor** toggle at the bottom that lets Admin and Builder users switch between two views of the vault. Builder mode shows the full interface; Contributor mode shows a simplified view that mirrors what Contributor and Viewer role users see — useful for previewing the experience before assigning access to your team.

## When to Use This

Use this when you want to preview how a vault appears to Contributor or Viewer users, or when you want to work in a focused, simplified view that shows only the tables that have been enabled for Contributor access.

## Prerequisites

- Admin or Builder vault role (the toggle is only visible to these roles)

## Step-by-Step Instructions

### Step 1: Open a Vault

Navigate to the workspace dashboard and click on any vault to open it.

### Step 2: Locate the Toggle

At the very bottom of the left sidebar, you will see two options side by side: **Builder** and **Contributor**.

### Step 3: Switch to Contributor Mode

Click **Contributor**. The sidebar updates immediately:

- Only tables that have **Enable for Contributor Mode** checked are visible
- The Schema and Settings tabs are hidden
- The **+** button to add tables is removed

### Step 4: Switch Back to Builder Mode

Click **Builder** to return to the full view with all tables, all tabs, and all configuration options visible.

## What to Expect

| Feature | Builder Mode | Contributor Mode |
|---------|-------------|-----------------|
| All tables visible | ✅ | ❌ (only Contributor-enabled) |
| Schema tab | ✅ | ❌ |
| Settings tab | ✅ | ❌ |
| Add table (+) button | ✅ | ❌ |
| Data tab | ✅ | ✅ |

## Common Issues & Troubleshooting

| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| Toggle is not visible | Your role is Contributor or Viewer | The toggle is only shown to Admin and Builder users |
| No tables appear in Contributor Mode | No tables have been enabled for Contributor Mode | Go to each table's Settings tab and check **Enable for Contributor Mode** |
| Settings or Automations pages redirect when in Contributor Mode | Access to those areas is blocked in Contributor Mode | Switch back to Builder mode to access Settings and Automations |
